# CardioVision: AI-Powered Coronary Artery Disease Analytics & Prediction System

## Overview

CardioVision is an end-to-end Healthcare Analytics and Machine Learning project designed to analyze, predict, and visualize Coronary Artery Disease (CAD) risk using patient clinical data. The project combines Exploratory Data Analysis (EDA), predictive modeling, interactive dashboards, and automated reporting to support data-driven healthcare decisions.

## Problem Statement

Coronary Artery Disease is one of the leading causes of death worldwide. Early identification of high-risk patients can significantly improve treatment outcomes and reduce healthcare costs. This project aims to develop an intelligent system capable of identifying patients at risk of heart disease using clinical indicators.

## Objectives

* Analyze heart disease patient data.
* Identify major clinical risk factors.
* Build machine learning models for disease prediction.
* Generate risk probability scores.
* Categorize patients into Low, Medium, and High Risk groups.
* Create interactive visualizations and dashboards.
* Produce executive-level healthcare reports.

## Dataset

The project uses the Heart Disease dataset containing clinical and demographic patient information such as:

* Age
* Sex
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol
* Fasting Blood Sugar
* Maximum Heart Rate
* Exercise-Induced Angina
* ST Depression
* Number of Major Vessels
* Thalassemia
* Target (Heart Disease Presence)
